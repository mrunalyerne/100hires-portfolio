## Issues I Ran Into & How I Solved Them

**Issue 1 — Claude Code Required Paid Subscription**
When I attempted to log in to the Claude Code extension, I received a message stating that Claude Max or Pro subscription is required. I researched the requirement, understood it, and upgraded to Claude Pro to complete the login successfully. This taught me that some AI tools have tiered access and it is important to verify tool requirements before starting a task.

**Issue 2 — Git Not Installed on Computer**
When I attempted to clone my GitHub repository using the git clone command in Cursor's terminal, I received an error: "git is not recognised as the name of a cmdlet." I searched for the solution, found that Git needed to be installed separately on Windows, downloaded it from git-scm.com, installed it with recommended settings, restarted Cursor, and successfully ran the clone command.
